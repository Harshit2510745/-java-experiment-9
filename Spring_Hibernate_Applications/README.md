# Spring & Hibernate Applications

This repository contains three Java-based examples demonstrating **Spring Framework** and **Hibernate ORM** concepts.

## 📘 Part A: Spring Dependency Injection (Java-based Config)
Demonstrates Spring's **Dependency Injection (DI)** using annotations and Java config (no XML).

## 🧮 Part B: Hibernate CRUD Operations
Implements basic **Create, Read, Update, Delete (CRUD)** for a `Student` entity using Hibernate ORM and MySQL.

## 💸 Part C: Spring + Hibernate Transaction Management
Simulates a simple banking system for **transactional money transfers** using Spring’s `@Transactional` with Hibernate ORM.
